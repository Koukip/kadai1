# LED_Device_Driver
2021年度のロボットシステム学の課題1で作成して改良を加えたデバイスドライバです。

# 動作環境
　・Raspberry Pi 3 Model B
 
  ・OS:ubuntu 20.04 server
  
# 使用したもの
　・Raspberry Pi 3 Model B × 1
 
  ・ブレッドボード × 1
  
  ・LED × 3
  
  ・ブザー × 1
  
  ・抵抗220Ω × 4
  
  ・ジャンパー線 (オス-メス) × 8
